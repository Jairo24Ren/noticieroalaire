Introducción

El documento propone la creación de una página web para el noticiero "Noticiero al Aire". Una página web donde en ella se incluirá contenido multimedia en formato HTML que encripta datos. Se espera proporcionar a los consumidores un mayor acceso y mejor organización de la información, así como contenidos audiovisuales. La web busca construir credibilidad para el noticiero, facilitar la interacción del usuario y mejorar la experiencia de búsqueda de información. Se espera que el sitio web atraiga más usuarios gracias a las formas de interacción y provoque en ellos el deseo de explorar más contenido, aumentando su satisfacción y evitando el estrés al navegar. Además, la página web estará optimizada para una carga rápida, lo que ayudará a crear una relación sólida y positiva entre el noticiero y sus usuarios. La web permitirá un acceso más amplio a la información específica del noticiero a través de enlaces a diversos contenidos, ofreciendo una alternativa distinta a los libros.

Objetivos

Objetivo General

Desarrollar una página web fácil de usar para que sus consumidores tengan una mejor experiencia.

Objetivos Específicos

  • Diseñar una página web interactiva con diferentes espacios para que el consumidor las explore de forma interactiva
  • Asegurar que la página web sea completamente funcional y su navegación sea fácil en dispositivos móviles.

Descripción del Proyecto

Nuestro proyecto será la creación de la página web para “Noticiero al aire” será un espacio digital en el cual todos los usuarios encontrarán diferentes secciones de información nacional e internacional, se dividirá en varios apartados que los dirigirán a secciones como, noticias nacionales, internacionales, clima, espectáculos, deportes.

Misión: En nuestro “Noticiero al aire” queremos brindarte una información rápida y confiable, para que siempre estés enterado de lo que ocurre tanto a nivel nacional e internacional, de estilo periodístico transparente.

Visión: Ser reconocidos por el profesionalismo periodístico, con respecto a los cambios que el país enfrenta, mantener siempre la postura de integridad y honestidad al informar a nuestra audiencia.

Plataformas de Creación

● Identificación de posibles plataformas de creación de sitio web

Como equipo decidimos que WordPress es la plataforma indicada para la creación de nuestra página web, ya que no necesariamente se necesita ser un desarrollador experimentado, como principiantes también podremos hacer uso de la plataforma, una de sus ventajas es que permite añadir imágenes, vídeos y audios en alta calidad para que los usuarios tengan una mejor experiencia e interactividad con la página web, desde nuestro dispositivo móvil podemos hacer uso de la plataforma.

Squarespace en esta plataforma se encuentran muchos diseños de plantillas, para crear la página para tu negocio o emprendimiento, se cree que su uso es más rápido y eficaz que wordpress, con la condición de que desde el inicio del proceso para tu página de negocio o emprendimiento se debe aceptar un plan de pago y al desear cambiar el uso de la plataforma fácilmente se pueden exportar los datos para wordpress y seguir el proceso desde ahí para la página web.

Shopify desde un inicio de crear una página para tu negocio te solicita un método de pago, no es 100% segura ya que al dejar de pagar una mensualidad estás en peligro de perder tus datos porque ya no te pertenecen.
